### Step 1: Create Venv
```bash
python3 -m venv myenv
```

### Step 2: Activate
- Linux/macOS: `source myenv/bin/activate`
- Windows: `myenv\Scripts\activate`

### Step 3: Deactivate
```bash
deactivate
```
